HRC Cutover Runtime

Broker-only runtime admission

Legacy fallback paths were removed or fail-closed, with headless and interactive starts routed through HarnessBrokerController.

  • Non-broker live runtimes are not reused for harness input.

  • Production references to the old exec launch surface were removed.
HRC Lifecycle Projection

  • Stale generation/attempt events are retained as evidence, not active state.
